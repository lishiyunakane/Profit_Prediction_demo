Welcome to the UI for our predictive model designed to forecast the profit margin of running projects using Excel Marco. To get started, simply upload the project reports you want to analyze, downloaded from task-hub. Please note that the file format must be Excel files. The 'Choose an Excel File' button below is where you can upload your file. We appreciate your participation and hope our predictive model can provide valuable insights for your projects.

Note: You are using a demo version designed to showcase the analytical process. The file for analysis has been pre-selected and represents a subset of the dataset. Simply click "Upload" to view the simulated analysis results.
